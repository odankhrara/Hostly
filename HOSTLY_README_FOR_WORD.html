What is Hostly?
A simple property rental platform where people can rent out their properties and travelers can find places to stay. Think Airbnb but simpler!

  • Database: MySQL to store all user and property information
  • AI Service: Python service to help with smart recommendations

üîß How to Set It Up

1. Install Dependencies

Make sure you have Node.js, Python, and MySQL installed on your computer.

2. Set Up Database

Create a MySQL database called "hostly" and run the setup scripts.

3. Start the Services

Run these commands in separate terminal windows:

# Backend (Node.js) cd backend && npm run dev # Frontend (React) cd frontend && npm run dev # AI Service (Python) cd agent-service && python -m uvicorn app.main:app --reload

üåê How to Use

  1. Open your browser and go to http://localhost:5173
  2. Sign up for an account (choose owner or traveler)
  3. If you're an owner, add your properties
  4. If you're a traveler, search for properties and book them
  5. Use the AI assistant for help and recommendations

üîê Security Features

üìä Database Tables

We have these main tables:

  • properties: Stores property details (name, location, price, etc.)
  • bookings: Stores booking information (dates, guests, status)
  • favorites: Stores user's favorite properties
